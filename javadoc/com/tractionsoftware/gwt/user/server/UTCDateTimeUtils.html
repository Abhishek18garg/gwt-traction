      • getDateValue

        public static final Date getDateValue(TimeZone zone,
                Long dateBoxValue,
                Long timeBoxValue)
        Returns the Date for the values of the UTCDateBox and UTCTimeBox which were edited in the specified TimeZone.
        Parameters:
        zone - The TimeZone in which the Date was edited.
        dateBoxValue - The value of the UTCDateBox control.
        timeBoxValue - The value of the UTCTimeBox control.
        Returns:
        The Date that has been selected from the controls or null if the supplied dateBoxValue is null. If the timeBoxValue is null, midnight is returned.

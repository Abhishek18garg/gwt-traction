Class UTCDateTimeRangeController

  • java.lang.Object
    • com.tractionsoftware.gwt.user.client.ui.UTCDateTimeRangeController

  • public class UTCDateTimeRangeController
extends Object
    The date/time range controller just manages existing controls and is not a control itself. It ensures that a range is consistent (start < end) and provides some conveniences like automatically adjusting the end time when the start time changes. These behaviors are consistent with other date/time range controls in other commonly used software.

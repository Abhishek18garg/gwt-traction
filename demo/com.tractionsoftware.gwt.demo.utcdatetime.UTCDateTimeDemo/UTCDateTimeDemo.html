The UTCDateTimeRangeController ensures that the range is a proper range. See the UTCDateBox and UTCTimeBox demos for specific details on those controls.

The UTCDateTimeUtils class is provided for use on the server to allow conversion to a Date in a known TimeZone.

Note: These demos use the "en" locale with default date/time formats, but other locales and formats can be specified.
